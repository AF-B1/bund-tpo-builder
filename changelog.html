Release history

v1.1.0 13 Jul 2026

Custom session open price — set where period A starts before you build.

Added

  • Price input in column 1 when the grid is empty (e.g. 125.50)
  • Press Enter to lock open price, update the ladder, and place A

Changed

  • Reset and empty grid return to the price input
  • Last chosen open price kept as the default in the input
v1.0.0 13 Jul 2026

First public release — teaching tool for keyboard-built Market Profile / TPO sessions.

Added

  • Keyboard TPO builder for Bund futures (125.50, 0.01 tick)
  • Split view: periods A–S, 07:00–16:30
  • Full profile: POC, 68.8% value area, Initial Balance, close
  • Three-column layout with vertical session progress
  • Live print, time, and price status + teaching guide

Layout

  • Orange IB bar (A+B) left of letters
  • Red close triangle right of letters, pointing in
  • POC: magenta row, white letters

Delivery

  • Live link via GitHub Pages
  • Standalone zip for local Windows use
